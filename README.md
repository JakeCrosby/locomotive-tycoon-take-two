# locomotive-tycoon-take-two

## Game Script Overview

The game is a highly detailed 3D locomotive design and business simulator spanning **1804 to 1990**.

## Core Gameplay Loop

- Start in **1804** with access to your workshop.
- Design and build custom locomotives using era-appropriate technology.
- Take contracts, source materials, manufacture locomotives, and ship them.
- Test locomotive designs and iterate as technology advances.
- Progress time using a **Year Up** system (single year or multiple years if affordable).

## Locomotive Designer Requirements

Players can design custom steam and diesel locomotives with interactive part placement and movement. Configurable parts include:

- Cylinder size
- Number of cylinders
- Cylinder location
- Boiler pressure
- Boiler type
- Firebox type
- Firebox size
- Boiler size (length and width)
- Frame type
- Number of wheels
- Era-based wheel arrangements with custom arrangements
- Valve gear
- Wheel size
- Fuel type

## Facilities and Systems

### Workshop
- Available from 1804.
- Primary location for designing and building locomotives.

### Contracts
- Accept work from customers such as mining companies and railroads.
- Some contracts require reproducing specific historical locomotives.

### Warehouse
- Ship finished locomotives.
- Order and manage materials needed for construction.

### Museum
- Contains **350 historical locomotive designs** with informational entries.
- Supports contract tie-ins where clients request museum locomotives to be recreated.

### Testing Area
- Dedicated area for trying locomotive performance before delivery.

## Time and Progression

- Timeline runs from **1804–1990**.
- Technology and available wheel arrangements are constrained by era.
- Year progression is player-controlled via **Year Up** and gated by in-game finances for multi-year skips.

## Visual and Interaction Requirements

- The game is **3D-based** and **highly detailed**.
- The locomotive designer is interactive, allowing players to move parts around during design.
